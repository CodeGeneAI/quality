# Security Policy

## Supported Versions

The latest minor release of `@codegeneai/quality` receives security patches. Older minors do not.

| Version | Supported |
| ------- | --------- |
| 2.2.x   | ✅        |
| < 2.2   | ❌        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security-sensitive reports.

Open a private advisory instead: **https://github.com/CodeGeneAI/quality/security/advisories/new**

Include:

- A clear description of the issue and its impact.
- Reproduction steps or a proof-of-concept config.
- The version of `@codegeneai/quality` and your Bun version.
- Any suggested mitigations or fixes, if you have them.

You can expect:

- Acknowledgement within a few business days.
- A coordinated disclosure timeline once we agree on a fix window.
- Credit in the release notes if you'd like it.

Thanks for helping keep `@codegeneai/quality` and its consumers safe.
